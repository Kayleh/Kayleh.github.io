创建标签云页面

添加一个标签云页面，并在菜单中显示页面链接。

新建一个页面，命名为 tags 。命令如下：

hexo new page “tags”

编辑刚新建的页面，将页面的类型设置为 tags ，主题将自动为这个页面显示标签云。

type: “tags”

注意：如果有启用多说 或者 Disqus 评论，默认页面也会带有评论。
需要关闭的话，请添加字段 comments 并将值设置为 false，如：

comments: false
